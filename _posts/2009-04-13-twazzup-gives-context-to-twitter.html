Twitter search is becoming one of the tools I use most for research. But Twitter search leaves a big gap. It lacks context. Twazzup provides that context with the “trackback” style feature that provides posts and relevant tags about the search. It is not complete but there is enough to give search results some relevance for real-time Twitter activity.
